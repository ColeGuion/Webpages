1Basic Grid Layout

A simple grid with fixed columns and rows. Perfect for equal-sized card layouts and basic structures.

📄 HTML Structure 
<div class="grid-basic">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item">Item 4</div>
  <div class="grid-item">Item 5</div>
  <div class="grid-item">Item 6</div>
</div>
🎨 CSS Styling 
.grid-basic {
  display: grid;
  /* Create 3 equal columns */
  grid-template-columns: repeat(3, 1fr);
  /* Create 2 rows of 100px height */
  grid-template-rows: repeat(2, 100px);
  /* Add spacing between grid items */
  gap: 10px;
}

.grid-item {
  background: #667eea;
  display: flex;
  align-items: center;
  justify-content: center;
}

2Grid Template Areas

Named grid areas make complex layouts intuitive and responsive. Perfect for traditional webpage structures.

📄 HTML Structure 
<div class="grid-areas">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="aside">Aside</div>
  <div class="footer">Footer</div>
</div>
🎨 CSS Styling 
.grid-areas {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: auto 1fr auto;
  /* Define named grid areas */
  grid-template-areas: 
    "header header header"
    "sidebar main aside"
    "footer footer footer";
  gap: 15px;
  height: 300px;
}

/* Assign elements to named areas */
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

3Implicit Grid with Auto-Fit

Responsive grids that automatically adjust to content and screen size. Great for image galleries and card layouts.

📄 HTML Structure 
<div class="grid-implicit">
  <div class="grid-item">Auto 1</div>
  <div class="grid-item">Auto 2</div>
  <div class="grid-item">Auto 3</div>
  <div class="grid-item">Auto 4</div>
  <div class="grid-item">Auto 5</div>
  <div class="grid-item">Auto 6</div>
  <div class="grid-item">Auto 7</div>
</div>
🎨 CSS Styling 
.grid-implicit {
  display: grid;
  /* Auto-fit creates responsive columns */
  grid-template-columns: 
    repeat(auto-fit, minmax(120px, 1fr));
  gap: 15px;
}

/* 
auto-fit: Fits columns to available space
minmax(120px, 1fr): Min 120px, grows to fill space
Items automatically wrap to new rows
*/

.grid-item {
  min-height: 80px;
  display: flex;
  align-items: center;
  justify-content: center;
}

4Complex Layout with Spanning

Advanced grid with items spanning multiple rows and columns. Perfect for magazine-style layouts and dashboards.

📄 HTML Structure 
<div class="grid-complex">
  <div class="grid-item big-item">
    Big Item
  </div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item span-2">
    Wide Item
  </div>
  <div class="grid-item">Item 5</div>
  <div class="grid-item span-3">
    Extra Wide
  </div>
</div>
🎨 CSS Styling 
.grid-complex {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(3, 100px);
  gap: 15px;
}

/* Span multiple columns */
.span-2 { 
  grid-column: span 2; 
}

.span-3 { 
  grid-column: span 3; 
}

/* Span both rows and columns */
.big-item { 
  grid-column: span 2; 
  grid-row: span 2; 
}

/* Alternative syntax:
grid-column: 1 / 3; (start/end lines)
grid-row: 1 / 3;
*/

💡Pro Tips & Quick Reference

Essential Grid Properties

🏗️ Container Properties

  • display: grid
  • grid-template-columns: 200px 1fr auto
  • grid-template-rows: 100px auto
  • gap: 20px (row + column gap)
  • grid-template-areas: "header main"
  • justify-items: start | center | end
  • align-items: start | center | end

📦 Item Properties

  • grid-column: 1 / 3 or span 2
  • grid-row: 2 / 4 or span 2
  • grid-area: header
  • justify-self: start | center | end
  • align-self: start | center | end
  • place-self: center (align + justify)

🚀 Responsive Grid Patterns

Auto-fit columns:
repeat(auto-fit, minmax(200px, 1fr))
Fractional units:
1fr 2fr 1fr (25% 50% 25%)

📱 Media Query Example

/* Desktop: 3 columns */
.grid {
  grid-template-columns: repeat(3, 1fr);
}

/* Tablet: 2 columns */
@media (max-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Mobile: 1 column */
@media (max-width: 480px) {
  .grid {
    grid-template-columns: 1fr;
  }
}
