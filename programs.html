CSS CodeDescription
:has() — Parent Selector

Style an element if it contains certain child/descendant elements.		 /* Select divs that contain an image */ div:has(img) { border: 2px solid blue; padding: 10px; } /* Select list items that contain a link */ li:has(a) { font-weight: bold; } /* Select h2 if the card has an image */ .card:has(img) > h2 { color: red; } /* Select form groups that contain invalid inputs */ .form-group:has(input:invalid) { border-left: 3px solid red; }
:is() — Simplify Complex Selectors

Takes a list of selectors and matches any of them, reducing repetition.		 /* Applies styling to headings, paragraphs, and .active list items */ :is(h1, h2, h3, p, li.active) { color: blue; }
:not() — Exclude Elements

Excludes elements from a selector.		 /* Style all list items except .highlight */ li:not(.highlight) { color: gray; }
